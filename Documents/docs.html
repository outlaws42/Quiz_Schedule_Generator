General synopsus

This program takes a list of teams and creates a schedule divided by rooms.
There are 3 teams in each room. The quiz length is figured at 20 min. This was designed
to fit the layout of our regional Bible quizzing meets for the Church of God General Conference.

Reason for this project

The reason I created this was to auto generate a randomized schedule for our annual quiz meets, and
so I wouldn't have to do it by hand. I also enjoy programming, so this allows me to polish my skills.

Requirements

For Installing
You will need the Python (3.4) and GTK (>= 2.24) libraries and their dependencies (Pango, GLib). Your distro (Linux) should provide packages for these, usually installed by default.
For Windows, you can download an installer from the website which bundles these libraries. The Windows version marked "portable" includes everything.

Microsoft Excel
Although this program doesn't require Excel to create the schedule, it does create it in Excel format (.xlsx). You will need a program that can read this type of file to be able to open it.
Here are some free options if you don't already have Excel. LibreOffice - You can read and write Excel docs, Microsoft Excel viewer - You can view and print from the viewer, but you can't edit.

Adobe PDF
As of version 1.1.0, the program can generate the schedule in PDF format. All you really need is one of the many pdf readers to print the schedule. Excel is not needed to print a schedule.

Menus

File >> Generate As Excel = This will generate the schedule in Microsoft Excel format.
File >> Generate As PDF = This will generate the schedule in Adobe PDF format.
File >> Settings = This opens the settings dialog.
File >> Exit = Close the program.

Help >> Documentation = Opens this document.
Help >> About = Info about the program.


Schedule Generate

When clicking either the buttons or the "Generate As" from the file menu, you will be presented with a save dialog
that you can save the schedule as Excel or pdf format. You can then choose a name and location for the schedule. Once you click OK, a
summary of what the schedule will contain pops up. Once you click OK on the summary dialog, you can browse to the location where
you saved the file. If the option "Open quiz schedule after creation" is checked in the settings dialog, the file you just saved
will be opened in the default application for Excel or PDF files.

Settings

Open quiz schedule after creation

If checked, the schedule will auto open in the default application that handles .xlsx or .pdf files after it is generated.
Note: xlsx files are Microsoft Excel files. If you don't have Excel, you can download LibreOffice or install the Excel viewer.
You can download Adobe acrobat reader or any other application that will read pdf documents.

Show quiz schedule title

If checked, you can put a text title to the schedule.

Quiz Schedule title

The title text you want to show on the schedule.

Number of quizzes before lunch

This is the number of quizzes you want before lunch: range is 4 to 7.

Number of quizzes after lunch

This is the number of quizzes you want after lunch: range is 4 to 7.

Schedule start time

This is time the schedule will start. You can choose from 09:30 or 09:40.

Schedule lunch length

This is length of the lunch break. You can choose from 40min or 60min.
